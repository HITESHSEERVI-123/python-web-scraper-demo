# Python Web Scraper Demo

This repository contains a Python script that scrapes data from a website and exports it into a CSV file.

## Features
- Extracts product name and price
- Cleans raw text data
- Saves output to CSV
- Uses Selenium and BeautifulSoup

## Technologies Used
- Python
- Selenium
- BeautifulSoup
- Pandas

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python scraper.py

## Output
A CSV file containing structured product data.
