# Python Web Scraper Demo

This project demonstrates a simple Python script for scraping data from a website and exporting it into a CSV file.

## Features
- Extracts product name and price
- Cleans raw text data
- Saves output to CSV
- Uses Selenium and BeautifulSoup

## Technologies
- Python
- Selenium
- BeautifulSoup
- Pandas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python scraper.py

## Output
A CSV file containing extracted product data.
